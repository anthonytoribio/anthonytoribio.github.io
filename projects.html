PROJECTS

Youtube Fast Forward

As a team, we created an Andorid app that let's the user create a Youtube Playlist that synced to their Youtube account. They can delete existing playlists, add videos to playists, delete videos from playlists. Users also have the option to auto-create playlist based on the keyword they enter. The app was created using Andoid Studio and Kotlin. We utilized several views, like Recycler views, and created our own HTTP handler to handle API calls to the Youtube Data 3.0 API.

MLB Daily Home Runs

A website that uses the MLB Data API to showcase the number of homeruns hit the day the user visits the website and the player that hit the homerun(s). The website was built using HTML, CSS, Javascript, and Django.
